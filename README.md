<h1 align="center">House Price Prediction using Machine Learning</h1>

<h2>Project Overview</h2>
<p>
This project predicts house prices using Machine Learning techniques. It includes data preprocessing,
exploratory data analysis (EDA), feature engineering, and regression model building.
</p>

<h2>Objective</h2>
<ul>
  <li>Analyze housing dataset</li>
  <li>Identify key features affecting house prices</li>
  <li>Build regression models</li>
  <li>Predict house prices accurately</li>
</ul>

<h2>Workflow</h2>
<ol>
  <li><b>Data Collection</b> – Load dataset</li>
  <li><b>Data Preprocessing</b> – Handle missing values, encoding</li>
  <li><b>EDA</b> – Visualization and correlation analysis</li>
  <li><b>Feature Engineering</b> – Select important features</li>
  <li><b>Model Building</b> – Apply ML algorithms</li>
  <li><b>Evaluation</b> – MAE, MSE, R² Score</li>
  <li><b>Prediction</b> – Predict house prices</li>
</ol>

<h2>Technologies Used</h2>
<ul>
  <li>Python</li>
  <li>Jupyter Notebook</li>
  <li>Pandas</li>
  <li>NumPy</li>
  <li>Matplotlib / Seaborn</li>
  <li>Scikit-learn</li>
</ul>
